三种常见模式

  • 命令或普通(Normal)模式：可以实现移动光标，剪切/粘贴文本，主要用于查看文件
  • 插入(Insert)或编辑模式：主要用于修改文件
  • 扩展命令(extended command)或命令(末)行模式：保存，退出等

注意：末行模式 -> 普通模式 中的Enter可以用Esc

查看文本文件内容

  • cat [-A <显示所有控制符>] [-b <对所有非空行编号>] [-s <压缩连续空行成一行>] [-n <对所有行编号>]
  • nl = cat -b 显示行号
  • tac 逆向显示文本内容
  • rev 将同一行的内容逆向显示
